动态链接库  
===================================  
  动态库已改成共享库, 可以真机模拟器使用
  
使用  
-----------------------------------  

### LinkOne  
  编译运行LinkOne , 在LinkOne/Products 下找到LinkOne.framework 
  
### LinkMain   
  运行LinkMain ,会在控制台显示Document的路径 , 把LinkOne.framework放到 Document文件夹下即可
### 按钮
   界面的两个按钮分别是进入有xib的控制器和没有xib的控制器  
  
### 打开方式    
    有两种打开方式, 1.dlopen  2.NSBundle
